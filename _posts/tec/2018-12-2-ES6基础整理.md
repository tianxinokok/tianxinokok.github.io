### 一、let命令基本用法

let用来声明变量，跟var的区别是let只在所在的内码块内有效
```
{
  var a = 10;
  let b = 10;
}
console.log(a) // 10
console.log(b) // b is not defined
```
把let用在for循环中很合适
```
for(var i = 0 ;i<5;i++){

}
console.log(i); //5
for(let j = 0 ;j<5;j++){
	
}
console.log(j); //j is not defined
```
